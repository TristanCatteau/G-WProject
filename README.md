# G-WProject
It's a 2D game made on Unity, in 2022, based on the gameplay of Game & Watch.

### Modules used:
- Movement
- Coroutine
- Strength with Rigidbody
- Sound System
- Prefab
- Vector drawing

### Control
Only the arrow keys.
